# sample-portal

Search for samples, add collection date and notes

## Local Development

_Start the backend_
Enter the backend directory and start the venv

```
cd backend
. venv/bin/activate
```

Run the Flask app

```
flask run
```

_Start the frontend_
Enter the frontend directory and install modules

```
cd frontend/sample-portal
npm install
```

Run the Next.js app

```
npm run dev
```
